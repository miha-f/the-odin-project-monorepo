# Binary search tree

Goal: Implement binary search tree in JS.

Tech: JS

[Live preview](https://miha-f.github.io/the-odin-project-monorepo/binary_search_tree/)

## Run project
```bash
node index.js
```

or open `index.html` in browser and check developer console for output.
