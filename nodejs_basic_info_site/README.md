# NodeJS basic info site

Goal: Implement basic webserver using Node (later I added implemenation in Express.js).
  
Tech: JS, Node, Express.js

## Run project
### Node
```bash
cd node
node index.js
# open on web: localhost:8080
```

### Express
```bash
cd express
npm install
node index.js
# open on web: localhost:8080
```
