# Shopping cart

Goal: Implement shop, user external api to get fake items ([Fake store API](https://fakestoreapi.com/)). Implement working shopping cart (updates when new item is add) and checkout page.
  
Tech: React, Vite

## Run project
```bash
npm install
npm run dev
```
