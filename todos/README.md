
# Todos

Goal: Create a todo app - create todos, mark their status, categorize todos under different projects.

Tech: HTML, CSS, JS

[Live preview](https://miha-f.github.io/the-odin-project-monorepo/todos/dist)

## Run project
```bash
npm install
npx webpack
open ./dist/index.html
```
