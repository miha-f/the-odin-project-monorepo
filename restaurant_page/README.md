# Restaurant page

Goal: Build a restaurant page. Main goal is to manipulate DOM via JS (create pages, handle page switching).

Tech: HTML, CSS, JS

[Live preview](https://miha-f.github.io/the-odin-project-monorepo/etch_a_sketch)

## Run project
```bash
npm install
npx webpack
open ./dist/index.html
```
